PathFinder AI - Web AI Application

Quick Start:
1. Access the files 'main.py' and 'test_main.py' from backend folder 
2. Backend: cd backend && python main.py
3. Access the files 'App.css', 'App.js', 'index.css' and 'index-js' from frontend folder (inside backend folder)
2. Frontend: cd backend, cd frontend && npm start
3. Open http://localhost:3000

Requires: Python 3.8+, Node.js 14+, HF API key
